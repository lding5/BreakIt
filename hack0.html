Thoughts:

Overall, this was a very nice warmup to get me started and a lot more excited about what I'm about to do.

I'm pretty comfortable with web dev (HTML, CSS, JS/ JQuery), so it wasn't too frustrating to put this website together.

Deploying this though, was something that I learned to do this week. There are several different ways that I could've done this by, such doing this on an actual server or Heroku. But I don't have a personal server that I can use, maybe a raspberry pi project later on. And as for Heroku, I didn't go this route because I wanted someone to take care of the infrastructure for me right now, since I wanted to get this website up and running first. So later, I'll play with Heroku. So I ended up with Amazon AWS.

To be honest, when I first started to look at Amazon AWS, it was quite daunting. There's so many options and it was overwhelming. But thankfully, Amazon provided very helpful and easy-to-follow tutorial links. In the end, I got a S3 and Route 53 (took me a while to think of what I want to name my site) set up with multiple mouse clicks and tadah, there it is all up and running! So yeah, a pretty good experience with that so far.
